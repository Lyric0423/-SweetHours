# 义工时光 Sweet_Hours

用几小时，换一种人生。

在快节奏的城市里，我们总有一些无法分身的一小时，也总有一些渴望看看窗外的一小时。「义工时光」是一个轻量的互助社区：让需要帮助的人快速找到“搭把手”的邻居，让想体验生活的人，通过一次次微小的任务，走进别人的生活，收获报酬、信任和故事。

## 当前原型

这是一个无构建依赖的移动端静态原型，覆盖轻量版 PRD 的核心路径：

- 任务广场：搜索、标签筛选、任务卡片流、收藏入口。
- 任务详情：任务信息、发布者信用、地点、安全提示与申请入口。
- 发布任务：任务类型、标题、描述、时间地点、灵活报酬表单。
- 消息沟通：匹配后聊天、安全提醒、快捷发送动作。
- 我的任务：发布/申请状态视图。
- 个人中心：信用分、勋章、实名认证、身份切换、安全中心。

## 运行方式

直接打开 `index.html` 即可预览；也可以在仓库目录启动一个本地静态服务：

```bash
python3 -m http.server 4173
```

然后访问 `http://localhost:4173`。
